Crosshair Studio

A powerful desktop application for creating, customizing, and exporting gaming crosshairs

Advanced Crosshair Model

Our advanced crosshair model supports extensive customization options including:

Component-Based Design

Each crosshair is made up of individual components that can be customized separately:

  • Line components (top, right, bottom, left)
  • Circle components (inner and outer)
  • Dot component (center)
Styling Options

Each component can be customized with:

  • Individual colors
  • Opacity levels
  • Thickness adjustments
  • Outline thickness and color
  • Line cap styles (butt, round, square)
Positioning Control

Advanced positioning control for each component:

  • Precise offsets from center
  • Length and gap adjustments for lines
  • Radius settings for circles
  • Size customization for dots
Animation Effects

Bring crosshairs to life with dynamic animations:

  • Breathing effect
  • Pulsing effect
  • Blinking effect
  • Scaling effect
  • Rotation effect
  • Adjustable speed and intensity
Global Properties

Apply global settings to quickly adjust the entire crosshair:

  • Global scaling
  • Global opacity
  • Global color override
  • Descriptive naming
Export Options

Share your crosshair designs:

  • CSV export with all properties
  • Profile management
  • Game-specific configurations

Crosshair Model Definition

/// A model representing a customizable crosshair
class CrosshairModel {
  /// The unique identifier
  final String id;
  
  /// The name
  String name;
  
  /// The description
  String description;
  
  /// Global scale factor
  double scale;
  
  /// Global opacity
  double opacity;
  
  /// Global color - will be applied if individual components don't specify a color
  Color? globalColor;
  
  /// Top line component
  LineComponent topLine;
  
  /// Right line component
  LineComponent rightLine;
  
  /// Bottom line component
  LineComponent bottomLine;
  
  /// Left line component
  LineComponent leftLine;
  
  /// Outer circle component
  CircleComponent outerCircle;
  
  /// Inner circle component
  CircleComponent innerCircle;
  
  /// Center dot component
  DotComponent centerDot;
  
  /// Animation properties
  AnimationProperties animation;
  
  // ... methods not shown for brevity
}
